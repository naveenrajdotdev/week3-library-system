# Console-Based Library Management System

A simple Java console application to manage books and members in a library.  
This project demonstrates OOP concepts, file handling, and real-world logic implementation.

---

## Features

### Book Management
- Add new books
- View all books
- Search books by title
- Delete books

### Member Management
- Register new members

### Transactions
- Borrow books
- Return books
- Automatic due date (7 days)

### Library Insights
- View total books
- View borrowed books count
- View total members

### Data Persistence
- Stores data in local files:
  - `data/books.txt`
  - `data/members.txt`
- Automatically loads data on startup
- Auto-saves after every operation

---

## Technologies Used

- Java (JDK 17+ / compatible with Java 25)
- OOP (Encapsulation, Abstraction)
- File Handling (BufferedReader, BufferedWriter)
- Collections (ArrayList, List)
